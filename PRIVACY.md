# Privacy Policy

**colab-mcp** is a developer tool. It does not collect, store, or transmit any personal data to external servers.

- All communication happens locally between the Chrome extension and a server running on your own machine (`localhost:7823`).
- The extension reads cell content from open Google Colab tabs solely to execute commands you issue.
- No browsing data, notebook content, or user information is sent anywhere outside your machine.
- Google Drive access is handled directly by you via your own OAuth credentials.

This extension has no backend and no analytics.
