Privacy Policy — colab-mcp

colab-mcp is a developer tool that bridges Claude Code and Google Colab. It does not collect, store, or transmit any personal data.

What the extension does

The extension communicates exclusively with a server running on your own machine (localhost:7823). It reads cell content from open Google Colab tabs only to execute commands you issue locally.

Data collection

None. No analytics, no telemetry, no external servers. All communication stays between your browser and your local machine.

Third parties

No user data is shared with or sold to any third party.

Google Drive

Drive access is handled by your own OAuth credentials. The extension itself does not interact with Google Drive.
